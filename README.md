## Descripción
Esta aplicación es un juego desarrollado por Miguel Méndez Murias en la asignatura de Computación Persona-Máquina de Ingeniería Informática del Software.

## ¿Cómo ejecutar?
1. Añadir al classpath el archivo lib/jhall.jar
2. Para jugar, ejecutar el Main e introducir en la pantalla principal los códigos que se encuentran en files/tickets.dat
